# AGPLv3 SSPL dual license
a dual license setup that uses AGPLv3 and SSPL license for your open source project needs

# how to use
modify the [LICENSE](https://github.com/eightynine77/AGPLv3-SSPL-dual-license/blob/main/LICENSE) file to your needs and after that, upload all three files (LICENSE, LICENSE-SSPL-1.0.txt, and LICENSE-AGPLv3.txt) to your github repo
